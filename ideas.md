# Feature Ideas

This document contains feature ideas for future brainstorming and development.

## Planned Features

### U-Untersuchungstermine
U examinations/appointment tracking - standard pediatric checkup schedule integration

### Graph für Gewicht
Weight graph visualization - display growth data over time in graphical format

### Gewicht mit Datum eingeben (Vergangenheit)
Allow entering weight measurements with historical dates (backfill past data)

### Checkliste Entwicklungsstörungen
Developmental disorder checklist - screening tool for identifying potential developmental concerns

### Major.minor.bugfix Versioning
Implement semantic versioning for releases

### Use Tags and Releases
Set up proper GitHub tags and release management

### Multiple Children
Support tracking multiple children in the same installation
